Installation

To install the SDK add the following script to the end of your body element

      <script type="text/javascript">
        (function () {
          var s = document.createElement('script');
          s.type = 'text/javascript';
          s.async = true;
          s.src = 'http://localhost:8080/fuul.local.js';
          s.setAttribute('project-id', 'cdf95efb-9693-4503-8133-4fc0bb3a1c04');
          var x = document.getElementsByTagName('script')[0];
          x.parentNode.insertBefore(s, x);
        })();
      </script>

Usage

Session Id

The SDK keeps track of the user's session id. The session id is generated by the SDK after the window loads..

When you send an event through the fuul function, the session id is automatically sent to the server.

E.g:

fuul('connect_wallet', { address: 'Oxcda72070e455bb31c7690a170224ce43623d0b6f' })

This will send an event to Fuul's Events API and will include the session id in the request.
